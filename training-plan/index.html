Training Plan
21-Week Half Marathon Build  ·  Jun 28 – Nov 15, 2026
Overview

Weekly mileage by day, building toward the Run to Feed the Hungry Half Marathon on Nov 15. The first two weeks are strides only while volume and the SI joint rebuild. From there, Wednesdays carry the quality session — intervals and hills alternate on Build weeks, while every Deload week gets tempo, the lowest-novelty stimulus. Wednesday structure: Strides only through week 2. From week 4 on, intervals and hills alternate on Build weeks while every Deload week gets tempo — the lowest-novelty session, easiest to control on a lighter week. Intervals are run at 10K effort (9:35–9:45/mi) — a notch easier than 5K effort. Hill reps stay in a 30–45 sec range throughout on whatever grade is available, progressing by rep count only, and are placed only on Build weeks given SI joint history. Race weeks override the structure with an easy run + strides — the race is that week's quality session. Taper week (Nov 8) holds intervals at the same effort but fewer reps. Paces are derived from a 9:25/mi 5K (Jun 3) and a ~12:00/mi easy baseline.

Strength Program

Two sessions per week, built around the SI joint history and slotted into the two open days — Tuesday and Friday — so neither lift day touches Wednesday's quality session or Saturday's long run. Click any Tuesday or Friday cell in the grid above for that week's exercises, sets, and reps.

Weeks 1–3 (Technique): Bodyweight and light dumbbell only. Both days run the same technique-first session — squat, hinge, push, pull, and core — to rebuild the patterns and confirm the SI joint stays quiet before any barbell load is added.

Weeks 4–14 (Foundation/Build): Barbell introduced. Tuesday alternates squat-emphasis and deadlift-emphasis week to week — one lift progresses in load while the other holds steady, so no single session stacks two near-max spinal-loading lifts. Friday is upper body and core only, with zero leg loading, because Friday sits the day before Saturday's long run and lower-body soreness peaks 24–48 hours after a session — exactly the window that would otherwise land on the long run.

Barbell deadlift escalation rule: Start at empty bar or bar + 10s for the first two weeks back, focused entirely on form. Add 10 lb in week 6 if pain-free, then 5–10 lb/wk after that — but only if the SI joint is silent the next day. Any twinge, tightness, or "off" feeling means dropping back to the prior week's weight, no exceptions. Elevating the bar on blocks or pins reduces range of motion and is worth using for the first several weeks if a rack is available.

Week 15 exception: Tuesday carries a 3-mile run that week instead of a rest/lift day, so Tuesday's strength session is dropped entirely rather than stacked with the run — Friday runs a single light upper/core session instead of two full sessions.

Weeks 16–19 (Peak): As running volume climbs to 22–26 mi/week, strength shifts to maintenance — same exercises and working weight, fewer sets. The goal is holding onto what's been built, not progressing further, while running takes priority.

Week 20 (Taper) and Week 21 (Race): One light Tuesday touch in taper week, no Friday session. Race week skips strength entirely — the race is that week's only output.
