Training Notes

Easy Runs (Mon)
Should feel fully conversational — full sentences without gasping. These build aerobic base and aid recovery. Never push these.
Mid Runs (Wed)
Moderate effort — comfortably hard, can speak a sentence or two. Think 60–70% effort. Builds mid-week endurance.
Quality Days (Thu)
Introduced Week 4. Includes intervals, tempo, and HMP work. Always warm up 1 easy mile first. On deload weeks, Thu is a short easy run (2–3 mi).
Long Runs (Sat)
The most important run of the week. Always easy pace — goal is time on feet. Practice race-day fueling and gear from Week 8 onward.
Heavy Lifting (Tue)
Always start with 20–25 min easy cycling warmup. Weeks 1–3: learn movement patterns with light load. Weeks 4–6: 3×5 linear progression. Weeks 7–11: 5×3 and 4×2 heavier loads. Olympic work (power clean / hang clean) builds rate of force development which directly benefits running economy. Add weight only when form is solid.
Bodyweight (Fri)
20–30 minutes, never exhausting. Focus on glutes, single-leg stability, core, and hip strength — the things that keep runners healthy. Friday sits between Thursday's quality run and Saturday's long run, so keep intensity low. Finish with mobility work.
Deload Weeks (3, 6, 9)
Running volume drops ~30–40%. Thursday is a short easy run (2–3 mi). Strength stays the same — the running reduction is the recovery mechanism. Do not skip deloads.
Taper (Weeks 15–16)
Running volume drops sharply. Strength reduces too — lower sets, lower weight, keep it snappy. No deadlifts in Week 16. Feeling sluggish or restless is normal. Arrive at the start line rested.
